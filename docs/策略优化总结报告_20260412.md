## 一、研究概述

本次优化研究的目标是在 J-Stock 系统中寻找最优的**入场策略 × 出场策略**组合，并对选定冠军策略进行精细参数调优后投入生产运行。

### 测试矩阵

| 入场策略 | 说明 | 配对出场族 |
|---|---|---|
| `MACDCrossoverStrategy` | MACD 金叉（基线） | MVX |
| `MACDPreCross2BarEntry` | MACD 预交叉 2K线 | MVX / MVXW |
| `MACDPreCrossHist2BarEntry` | MACD 柱状图 2K线 | MVX / MVXW |
| `MACDPreCrossHist3BarEntry` | MACD 柱状图 3K线 | MVX / MVXW |
| `MACDPreCross3BarEntry` | MACD 预交叉 3K线（含价格上涨条件） | MVX / MVXW |

**出场策略族：**
- **MVX** = MultiView Exit（标准版），核心参数 N/R/T/D/B
- **MVXW** = MultiView Exit Weighted（加权版），同族参数但含窗口权重衰减

每组合均经历 **N扫描 → R扫描 → T扫描 → Grid1(3×3) → Grid2(3×3)** 五轮逐步缩小的网格搜索，最终选出各组合冠军。

## 三、MVXW 冠军精细调优

选定 `MACDPreCross2BarEntry + MVXW` 为生产策略后，进行了四阶段精细网格搜索：

### 第二阶段：R×T 5×5 精细网格（25策略 × 5年 = 125回测）

R ∈ {3.2, 3.25, 3.3, 3.35, 3.4}，T ∈ {1.45, 1.5, 1.55, 1.6, 1.65}

**关键发现 —— T=1.45 阈值效应：**

| T值 | 亏损年数 | Target20 范围 |
|:---:|:---:|---|
| **T=1.45** | **0** | 92.96 ~ 93.39 |

| T=1.60 | 1 | 52.42 ~ 56.36 |

T=1.45 是一个**二元阈值**：所有 T=1.45 的变体均为 0 亏损年，而 T≥1.50 全部有 1 个亏损年。这导致 target20 出现约 30 分的断崖。

### 第四阶段：T 下界边沿验证

T=1.45 处于 Grid2 搜索空间的边界，需确认更低 T 值是否更优。

固定 N=5, R=3.35, D=21, B=20.0，扫描 T ∈ {1.3, 1.35, 1.4, 1.45}

**结论：** T<1.45 虽然也无亏损年，但 Target20 和累计收益均单调递减。T=1.45 确认为真正最优点，非边界伪影。

### 精细调优最终结果

> 调优的核心收获在于**消除了唯一的亏损年**，使 Target20 评分从中等水平跃升至顶级水平。

### 📊 关键洞察

- **数据修正影响显著**：排名变动幅度大（最大上升7位，最大下降6位），旧生产参考策略出现亏损年
